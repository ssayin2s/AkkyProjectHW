# AkkyProjectHW

Bu proje, MongoDB ve Kafka'nın entegrasyonunu göstermek için geliştirilmiştir.

## Kullanılan Teknolojiler

- Docker
- MongoDB
- Apache Kafka
- Java
- Kafka Client API
- MongoDB Java Driver
